# Mean-todo
Todo application example with Express+Mongojs+Angular2+Nodejs 

## １．step1
```
git clone https://github.com/nochi0105/mean-todo-app.git && cd mean-todo-app
```

## ２．step2
```
npm install -g @angular/cli@latest
```

## ３．step3
```
npm install && npm run start
```
## ４．latest
http://localhost:3000
